参加天体疗愈的疑虑

Q01：为什么要采取"天体（零束缚）"的形式进行疗愈？
皮肤是人体最大的感官器官。在苏眉的理念中，衣物不仅是物理遮盖，更是社会身份的枷锁。褪去衣物是为了彻底卸下"社会标签"，让身体直接与空气、阳光、精油和声音频率接触，实现真正意义上的感官觉醒与自我接纳。
Q02：天体疗愈会涉及到性吗？
绝对不会。苏眉倡导的是"非性化（Non-sexual）"的自然主义。我们所有的课程（瑜伽、冥想、芳疗）均建立在专业的疗愈框架下。任何带有性暗示的行为或言语在我们的场域内都是被严格禁止的，违者将被立即取消会员资格。
Q03：我对自己身材不自信，会感到尴尬吗？
这正是我们存在的意义。在苏眉，没有"标准身材"，只有"生命个体"。你会发现，当你置身于一个全然接纳、不予评判（Non-judgmental）的群体中时，那种对他人的羞涩和对自己的挑剔会迅速转化为一种前所未有的自由感。
Q04：如何保障参与者的隐私安全？
隐私是我们的生命线。
✓ 入场封手机：所有线下活动必须上交电子设备或封禁摄像头。
✓ 实名审核：所有成员均经过苏眉亲自审核。
✓ 私人场域：我们只在租赁的私人海滩、会所或工作室开展活动，严禁非相关人员进入。
Q05：为什么不接受单身男性参加？
为了确保场域内女性成员的绝对安全感与放松度。我们目前仅对女性、夫妇及情侣开放。这是为了建立一个低防御、高信任的能量圈层，让每一位女性都能在没有"被凝视感"的环境中全然绽放。
Q06：如果活动中身体产生了自然的生理反应怎么办？
身体是非常诚实的，我们要学会以平常心看待。在专业的疗愈引导下，参与者的注意力通常会聚焦在呼吸、能量流动或感官体验上。如果偶尔出现紧张或局促，我们会引导你通过深呼吸将其转化为内在的能量，无需为此感到羞耻。
Q07：参加天体海滩营队，会被晒伤吗？
我们非常看重身体的养护。我们会提供专业的防晒建议（物理避晒与天然植物护肤），并选择在紫外线温和的时段进行户外活动。尊重自然，也意味着学会与阳光和谐共处。
Q08：我需要提前做全身除毛（Body Grooming）吗？
完全不需要，除非你想这样做。我们鼓励你保持身体最原始、最舒适的状态。你的身体毛发是你生命力的一部分，在这里，不需要为了迎合任何人的审美而改变自己。
Q09：第一次参加，如果我还是想穿一点衣服可以吗？
我们尊重每个人的节奏（Timing）。在苏眉，我们从不强迫。你可以先选择半遮盖，或者穿着轻薄的纱巾。当你感受到环境的包容与安全时，你会自然而然地想要卸下最后一层束缚。
Q10：我来天体营时恰逢生理期，还能参加活动吗？需要注意什么？
天体营对女性生理期持包容态度，您完全可以正常参与活动。
✓ 卫生用品选择：使用卫生棉条、月经杯，外观上不会有明显痕迹
✓ 如需使用卫生巾：可以穿一条轻薄的底裤或沙滩短裤，这是天体营普遍接受的做法
✓ 更换处理：使用卫生间设施正常处理，废弃物请用小袋子密封后丢弃
✓ 游泳活动：暂停
Tips：天体营倡导的是身体自然与尊重，工作人员和其他参与者对此类情况非常理解。如有任何不适或需要，随时可向现场工作人员寻求帮助。
Q11：大家都裸着，个人及公共区域的卫生怎么保障？有什么礼仪需要遵守？
天体营有一套成熟的卫生礼仪规范，参与者需共同遵守：
✓ 就坐前：必须在座椅、躺椅、长凳上铺设个人毛巾
✓ 游泳：使用泳池后应先冲净身体再晒太阳或进入室内
✓ 公共设施：桑拿、蒸汽房等区域需遵守各场所的具体卫生规定
✓ 个人清洁：保持身体清洁、无异味、指甲修剪整齐、每次如厕后都冲洗下体
违反规则可能被工作人员提醒，多次违反或被要求离场。这是天体营最基本的卫生底线。
Q12：如何加入你们？
为了保证同频，我们采取先体验、后入会的模式。你可以先预约一次1.5小时的单项疗愈或天体瑜伽体验。在相互了解、能量契合的基础上，再申请成为年度会员，解锁更多私密营队名额。

实践自然美学的疑虑

Q1. 不穿内衣会不会胸部下垂？
恰恰相反。法国研究（Rouillon，2013）发现长期穿胸罩反而弱化胸部韧带支撑力。不穿内衣让胸部肌肉自主发力，长期反而更有弹性。下垂的主因是年龄、重力和遗传，不是braless。
Q2. 出门不穿内衣，别人会不会看？
刚开始会有一点自我意识，但那是你的感受，不是别人的眼神。大多数人根本不会注意。适应期大概1-2周，之后你会发现自己才是最在意的那个人。
Q3. 不穿内裤会不会不卫生？
反过来才是。棉质内裤吸湿但透气差，长时间穿容易滋生细菌。不穿内裤让私处保持干燥通风，妇科医生普遍认为睡觉不穿内裤对私密健康有益。
Q4. 不穿内裤，分泌物怎么办？
正常的分泌物是无害的，这是身体自我清洁的机制。选择宽松棉质或丝质裤子可以吸收日常分泌。如果分泌物异常（颜色、气味），那是去看医生的信号，不是穿内裤能解决的问题。
Q5. 裸睡会着凉吗？
身体会自动调节体温，裸睡时核心体温会略降，这正是促进深度睡眠的机制之一。盖好被子就不冷，反而很多人裸睡后说睡得更深。
Q6. 裸睡对皮肤好还是不好？
好。睡衣和松紧带会压迫皮肤、阻碍血液循环。裸睡让皮肤整夜自由呼吸，减少摩擦，有助于改善皮肤状态。
Q7. 和伴侣裸睡会不会变成「必须做什么」的压力？
裸睡是亲密，不是邀请。两个人舒服地在一起才是重点，身体的边界每个人都有权设定。健康的伴侣关系里，裸睡会增加亲密感和安全感，不会制造压力。
Q8. 这些做法是不是太「暴露」、不够自重？
这个逻辑值得反问：谁规定覆盖身体才叫自重？身体是自己的，照顾好自己的舒适和健康才是真正的自我尊重。这些都是私人空间的个人选择，和道德无关。
Q9. 我男朋友/老公会不会觉得奇怪或反对？
可以先从家里、睡觉开始，不需要一开始就宣布什么。很多伴侣接触之后反而觉得自然、甚至欣赏这份自在。如果伴侣完全无法接受你正常的身体选择，那是值得认真谈的关系议题。
Q10. 这样生活是不是很小众、很难被理解？
越来越多女性在悄悄这样做，只是不说出来。Braless在欧美早已是主流，裸睡在各国调查里比例都不低。小众的不是这件事本身，是愿意公开谈的人还不多——苏眉存在的意义就在这里。
Q11. 在家不穿衣服生活，不会觉得奇怪或不自在吗？
刚开始会有点不习惯，这很正常，因为我们从小被教导要「穿好衣服」。在家不穿适应期通常只需要几天。一旦习惯了，穿着衣服在家反而会觉得多余。很多人描述的感受是：轻盈、自由、像真正在休息。
Q12. 在家裸体，安全吗？有什么需要注意的？
完全安全，注意几件实际的事就好：
✓ 窗帘拉好——不是因为羞耻，是给自己一个无打扰的私人空间
✓ 开门前认清——快递、访客来时有个习惯性的「穿衣动作」
✓ 做饭注意——煎炒时穿条围裙保护皮肤，这是实际安全问题
✓ 室温舒适——身体裸露时对温度更敏感，保持让自己舒服的温度
其余时间躺着、工作、看书、瑜伽……完全不需要任何理由，就是舒服。
